## Summary

Set an explicit numbering value at the target item. Mid-sequence targets are atomically separated first.

- Operation ID: `lists.setValue`
- API member path: `editor.doc.lists.setValue(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a ListsMutateItemResult receipt.

| Field | Type | Required | Description |

| `target.kind` | `"block"` | yes | Constant: `"block"` |

