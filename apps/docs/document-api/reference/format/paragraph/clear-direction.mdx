## Summary

Remove explicit paragraph direction, reverting to inherited or default (LTR).

- Operation ID: `format.paragraph.clearDirection`
- API member path: `editor.doc.format.paragraph.clearDirection(...)`
- Mutates document: `yes`
- Idempotency: `conditional`
- Supports tracked mode: `no`
- Supports dry run: `yes`
- Deterministic target resolution: `yes`

## Expected result

Returns a ParagraphMutationResult; reports NO_OP if no direction is set.

| Field | Type | Required | Description |
| --- | --- | --- | --- |
| `target` | ParagraphAddress \| HeadingAddress \| ListItemAddress | yes | One of: ParagraphAddress, HeadingAddress, ListItemAddress |
